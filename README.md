Mini App – User Registration & Authentication
Project Overview

This project is a mini authentication system that allows users to register, log in, view their profile, and log out. The system is designed as a simple full-stack application and will be used as the basis for future implementation during face-to-face sessions.

This repository focuses on documentation, system design, and UI structure, with backend and mobile components prepared for later development.

Features

User registration

User login

Protected profile/dashboard page

User logout

Restricted access to protected pages when logged out
